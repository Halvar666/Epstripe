Epstripe 1.0.0a11 source and license notes
===============================================

Epstripe is a Winstripe/XP-style complete theme for Epyrus.
This clean alpha build was prepared by Halvar666 from the confirmed Epstripe 1.0.0a9 visual state.

Confirmed visual baseline included in this clean build
-----------------------------------------------------

- a4: Correct Tag / Back / Forward toolbar cells from Winstripe 1.9.9, with the proper source-to-Epstripe sprite mapping.
- a8: Increased #mail-bar3 vertical padding to 2px top and 3px bottom, making the mail toolbar 5px taller without shifting the whole toolbox upward.
- a9: Improved inactive selection readability in the folder tree and message/thread list.

Abandoned experiments intentionally not included
------------------------------------------------

- a5: Skin-side XBL appmenu label experiment.
- a6: Content overlay / XBL fallback appmenu label experiment.
- a7: .mail-toolbox margin-top experiment that shifted the toolbox upward instead of increasing usable vertical space.

Known external issue
--------------------

In some localized Epyrus language packs, the app menu button may still display "Thunderbird" instead of "Epyrus". The English UI already displays the correct text. This is caused by language-pack branding strings, not by the Epstripe theme.

License compatibility choice
----------------------------

The project uses MPL 2.0 as the package license because:

- ClassicTB2 for Epyrus uses an MPL 2.0 install manifest and includes older Mozilla/Thunderbird theme files with their original headers.
- xpDefault colored buttons is MPL 2.0.
- Very old Mozilla/Thunderbird/Winstripe-era tri-licensed files or assets are used under the MPL branch and, where needed, treated as compatible with MPL 2.0 without adding an "incompatible with secondary licenses" notice.
- No GPL 3.0-only, GPL-only, MIT/BSD/Apache-only, or Public Domain-only relicensing path is selected.

4. Thunderstripe 1.6 Windows Thunderbird theme

   License: MPL 2.0

Notable imported/modified areas
-------------------------------

- Main mail toolbar sprites, including Winstripe 1.9.9 Tag / Back / Forward cells and Epstripe green unread double-arrow corrections.
- Compose toolbar sprites, including TB52/Epyrus-compatible format-buttons.png from xpDefault.
- Address Book toolbar and icon assets.
- Folder/server/message/thread/attachment/junk/read/unread state icons.
- Preferences, S/MIME, Activity Manager and New Mail Account icons.
- Calendar/Lightning skin packages from xpDefault for XP-style Calendar, Tasks, Add as event and Add as task icons.
- Main toolbar geometry is inherited from Default Epyrus; no Winstripe/Thunderstripe toolbar background or window background texture transplant is used.
- #mail-bar3 has a small Epstripe-only vertical padding adjustment to give the toolbar more height.
- Inactive selected rows in folder and thread trees use dark text to remain readable when focus moves to another tree.
