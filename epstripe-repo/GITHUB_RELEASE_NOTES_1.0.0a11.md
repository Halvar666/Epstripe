# Epstripe 1.0.0a11

Initial public alpha build of Epstripe, a Winstripe/XP-style complete theme for Epyrus.

## Added

- New Epstripe identity, package metadata and add-on listing artwork.
- 32×32 add-on icon based on the Get Messages toolbar icon.
- Updated preview image for the Add-ons Manager / repository.
- Source and license tracking notes for ClassicTB2, Epyrus, Winstripe-era, Thunderstripe and xpDefault colored buttons sources.

## Changed

- Main mail toolbar uses Winstripe/Thunderstripe-style icons with corrected Winstripe 1.9.9 Tag / Back / Forward cells.
- Mail toolbar has slightly more vertical space for better icon readability.
- Inactive selected rows in the folder tree and message list use readable dark text.
- Calendar/Lightning toolbar icon coverage retained through the xpDefault colored buttons skin layer.

## Known issue

- In some localized Epyrus language packs, the app menu button may still show “Thunderbird” instead of “Epyrus”. This is a language-pack branding string issue, not a theme issue.

## Validation

- `unzip -t` passes for the packaged XPI.
