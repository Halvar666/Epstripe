# GitHub repository setup notes

Suggested repository name:

```text
Epstripe
```

Suggested description:

```text
Winstripe/XP-style complete theme for Epyrus.
```

Suggested visibility:

```text
Public
```

When creating the repository on GitHub, do not initialize it with README, .gitignore, or license, because this package already contains those files.

Suggested first commands:

```bash
git init
git add .
git commit -m "Initial Epstripe alpha"
git branch -M main
git remote add origin https://github.com/Halvar666/Epstripe.git
git push -u origin main
```

Suggested first release:

```text
Tag: v1.0.0a11
Title: Epstripe 1.0.0a11
Artifact: dist/Epstripe_1.0.0a11.xpi
```

Use `GITHUB_RELEASE_NOTES_1.0.0a11.md` as the release body.
